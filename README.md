# SUMO Real-Time Traffic Simulation

This project is a Java application for **real-time traffic simulation** based on **SUMO** (Simulation of Urban Mobility) and the Java API **TraCi**.  
The application is designed to connect to a running SUMO simulation, visualize traffic flow, calculate key figures, and provide export functions.

> **Note:** The project is still in the early stages of development

---

## Project objectives

- Connect to a running SUMO simulation
- Show an interactive map view with vehicles and traffic lights
- Control the simulation and more via GUI
- Compute traffic analytics, including:
  - Average vehicle speed
  - Total and stationary vehicle count
  - Vehicle density per edge
  - Congested edges (hotspots)
  - Travel time distribution
- Export results as reports (e.g. CSV, PDF)
